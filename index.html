About the project

Web develompment merujuk pada proses pembuatan, pembangunan, dan pemeliharaan situs web serta aplikasi web yang berjalan secara daring pada peramban (browser). Ini melibatkan berbagai tugas, termasuk desain web, pemrograman web, dan manajemen basis data. Web development dapat dibagi menjadi dua spesialisasi utama: pengembangan front-end dan pengembangan back-end.

Web ini dibuat untuk memenuhi Souvenir dari Cyber Academy oleh kelompok KONGO.
